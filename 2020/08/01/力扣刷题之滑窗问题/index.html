**632\. 最小区间**

你有 k 个升序排列的整数数组。找到一个最小区间，使得 k 个列表中的每个列表至少有一个数包含在其中。

我们定义如果 b-a < d-c 或者在 b-a == d-c 时 a < c，则区间 [a,b] 比 [c,d] 小。

示例 1:
输入:[[4,10,15,24,26], [0,9,12,20], [5,18,22,30]]
输出: [20,24]
解释: 
列表 1：[4, 10, 15, 24, 26]，24 在区间 [20,24] 中。
列表 2：[0, 9, 12, 20]，20 在区间 [20,24] 中。
列表 3：[5, 18, 22, 30]，22 在区间 [20,24] 中。

解题思路：

如果nums列表里的所有数的最大值为Max，最小值为Min，则题目找到的结果
一定是res <= [Min, Max]的。

如果有一个列表：

nums = [[-1,2,3],[1],[1,2],[1,1,3]]

其实，列表的集合为：

0: [-1, 2, 3]
1: [1]
2: [1, 2]
3: [1, 1, 3]

冒号前面的是nums列表里每个数组的索引，冒号后面是索引代表的数组。

与之相对的，也可以得到这样一个哈希map：

-1: [0]
 1: [1, 2, 3, 3]
 2: [0, 2]
 3: [0, 3]

即-1出现在0号列表中，1出现在了1,2,3号列表中，并且还在3号列表里
出现了两次。2出现在了0,2号列表中。3出现在了0,3号列表中。

根据题意，我们要得到一个区间，使得nums里的每个列表都有一个数，
在该区间内。也就是说，我们可以遍历哈希map的所有值，只要出现的值能够
涵盖nums的所有索引即可。

这是什么意思呢？
-1出现在了0号列表，1出现在1,2,3号列表，nums里的每个列表，都存在元素出现在区间[-1,1]里了。
所以符合题意的区间有[-1, 1],[-1,2],[-1,3],[1,2],[1,3]。但是题目要求最小区间，即区间长度最短，
长度相等时选左区间更小者。所以最终答案是[-1,1]。

哈希map里的每个key就是区间的左右端点的备选，设左区间是left，右区间是right。由于之前所述，则题目找到的结果
一定是res <= [Min, Max]的。在初始化的时候，left = Min（左端点从最小值开始搜索）。而right的搜索，到Max停止。
这就相当于我们直接在哈希map中搜索。

然后就涉及到滑窗问题，这里的滑窗，左指针就是left，右指针就是right，滑窗就是一个区间。在搜索中，
一面要找到包含了所有nums数组索引的滑窗，一面要使得滑窗最小。
前者需要右指针右移。后者需要左指针右移。当然不是想移就移的，需要符合条件。

开始搜索的时候，left和right在一起，都指向了哈希map的第一个key，也就是-1。然后遍历键-1的所有值，也就是[0]。滑窗还
没包含所有索引，right右移，指向了1，发现键1的所有值为1,2,3。这下刚好包含了所有的索引，记录下来，继续往下遍历。
万一还能找到更短的区间呢？

右移左指针，让滑窗内由不完全包含所有索引了，迫使右指针继续向前探索。只有找到更小的right - left才会更新
区间，这样就同时满足了最小区间的两个条件。

综上所述：

right右移当仅当滑窗内的value不能包含nums的所有区间。

left右移当仅当滑窗内的value包含了所有nums内的区间。

res更新当仅当当前窗口包含所有区间，并且[left, right]是更小的区间。

代码如下：


class Solution {
    public int[] smallestRange(List> nums) {
        int size = nums.size();                 
        int xMax = Integer.MIN_VALUE;           // 记录nums列表的最大值和最小值
        int xMin = Integer.MAX_VALUE;
        Map> indices = new HashMap<>();      // 数字:[索引数组]

        // 找到最大最小值，建立哈希Map
        for(int i = 0; i < size; i++){          // 遍历nums列表的内的每个List
            for(int x : nums.get(i)){                      // 遍历每个列表内的数字
                List list = indices.getOrDefault(indices.get(x), new LinkedList());
                list.add(i);                    // 把区间索引加到列表里去
                indices.put(x, list);
                xMax = Math.max(x, xMax);
                xMin = Math.min(x, xMin);
            }
        }

        int inside = 0;
        int left = xMin;
        int right = xMin - 1;
        int bestRight = xMax;
        int bestLeft = xMin;
        int[] freq = new int[size];     // 用来记录索引出现的次数

        while(right < xMax){            // 遍历这个哈希map
            // 右指针右移
            right++;                    // 加一
            if(indices.containsKey(right)){            // 包含这个key（nums包含这个数字）
                for(int x : indices.get(right)){          // 遍历这个key的所有索引
                    freq[x]++;
                    if(freq[x] == 1)    inside++;              // 滑窗内包含的数组个数加一
                }
                // inside == size时，尝试更新区间
                while(inside == size){
                    if(right - left < bestRight - bestLeft){
                        bestLeft = left;
                        bestRight = right;
                    }
                    if(indices.containsKey(left)){
                        for(int x : indices.get(left)){
                            freq[x]--;
                            if(freq[x] == 0)    inside--;
                        }               
                    } 
                    left++;
                }
            }

        }
        return new int[]{bestLeft, bestRight};
    }
}
